# Hangman
  this is Hangman game in command line
  
## Build
 1. get python 3.0 or above
 2. go to directory hangman in your cmd or terminal
 3. if you not have xmltodict please install first by pip3 or etc. 
 ```zsh
$ pip3 install xmltodict
```
 
for Windows
```zsh
$ py -3 hangman.py
```
for Mac and linux
```zsh
$ python3 hangman.py
```
